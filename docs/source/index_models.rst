Models
=======

This module provides the implementation and defintion of the models used in the Voice Lead Agent project. It includes the database models, and all the necessary classes and functions to interact with the database. The models are used to represent the data in the application, and they are responsible for defining the structure of the data, as well as the relationships between different entities.
