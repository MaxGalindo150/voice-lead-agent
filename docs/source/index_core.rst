Core
====

This module is the core of the Voice Lead Agent project, responsible for managing the brain of the agent, implementing the LLM (Language Model) integration, ASR (Automatic Speech Recognition), and TTS (Text-to-Speech) functionalities. It also handles the conversation management and the interaction with the database.
